Welcome to the official site for Everybody Edits Rewritten!

Everybody Edits Rewritten is a free-to-play, online multiplayer game that allows players to create and share their own worlds!

About running EER

To be as faithful to the original Everybody Edits as possible, Everybody Edits Rewritten is made in Adobe Flash. This means that to run the game, you need some way to open .swf files. Unfortunately, due to security vulnerabilities inherent to Flash, Adobe discontinued Flash Player in December 2020 and put in a lot of extra effort to make running .swfs difficult.

If you have doubts about the integrity of our code, feel free to take a look at its source code using a tool like FFDEC (JPEXS Free Flash Decompiler) before running it!

How to run the game

To run EER, you need the client (Everybody.Edits.Rewritten.swf) and some way to open it (Flash Player/Projector version 32). We provide download links for both on this site's Downloads page. Make note of where you download the client!

Once you've downloaded Flash Player and a copy of the client, open Flash Player, click File>Open in the top-left corner, click "Browse...", and select the .swf you downloaded. That's it -- you're ready to play!
